# 南林课程表 (NJFU Schedule)

>没办法，wakeup课程表吃相太难看了，广告比功能多。

因此而设计:面向南林教务系统课表导入、日常查看、自定义课程管理和桌面小组件展示场景开发。项目基于 [WakeupSchedule_Kotlin](https://github.com/YZune/WakeupSchedule_Kotlin) 的课程数据模型重新构建，


## 构建

1. 使用 Android Studio 打开项目。
2. 等待 Gradle Sync 完成。
3. 连接真机或启动模拟器。
4. 运行 app 模块。

## 导入原理

1. 通过南京林业大学统一认证系统进行 CAS 登录。
2. 按学校登录流程对密码参数进行加密处理。
3. 登录成功后访问教务系统课表页面。
4. 使用 Jsoup 解析 HTML 表格，提取课程名、教师、教室、周次、节次等信息。
5. 将不连续周次拆分为多条课程详情记录。
6. 写入本地 Room 数据库，并刷新课表和桌面小组件。

## 小组件

当前提供两类桌面小组件：

- **今日课程列表小组件**：展示当天课程列表，适合较宽区域。
- **今日课程 2x2 小组件**：紧凑展示今日课程，突出第一节课，并显示后续课程或地点信息。

小组件刷新机制：

- **系统定时刷新**：每 30 分钟由系统自动触发 `onUpdate()` 刷新。
- **系统广播触发**：监听 `ACTION_TIME_CHANGED`、`ACTION_DATE_CHANGED` 和自定义 `REFRESH_WIDGET` 广播，时间或日期变化时立即刷新。
- **应用内主动刷新**：导入课表、添加/编辑/删除课程后，通过 `refreshAll()` 主动推送更新到小组件。

## 说明

本项目仅用于学习和个人课表管理。账号密码仅用于登录南京林业大学统一认证和教务系统，课程数据仅保存在本地数据库中。
