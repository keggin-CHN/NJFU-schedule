# 南林课程表 (NJFU Schedule)

南京林业大学专属课程表 Android 应用，面向南林教务系统课表导入、日常查看、自定义课程管理和桌面小组件展示场景开发。项目基于 [WakeupSchedule_Kotlin](https://github.com/YZune/WakeupSchedule_Kotlin) 的课程数据模型重新构建，并针对南京林业大学教务系统进行适配。

## 功能

- **一键导入课表**：输入学号和统一认证密码，自动登录教务系统并解析本学期课表。
- **周视图课表**：按周展示课程，支持切换周数、显示当天列、区分本周课程和非本周课程。
- **节次与时间显示**：左侧时间轴同时显示节次、开始时间和结束时间。
- **自定义课程**：支持手动添加、编辑课程，可设置课程名、教师、教室、周次、日期范围、节次或具体时间。
- **课程颜色**：支持为自定义课程选择颜色，课表卡片和小组件会同步使用课程颜色。
- **冲突与同步处理**：同步教务课表时保留自定义课程，并提示课程变动和时间冲突。
- **全校课表查询**：提供教师课表、教室课表、课程课表、班级课表查询入口。
- **桌面小组件**：支持今日课程列表小组件和 2x2 今日课程紧凑小组件。
- **背景设置**：支持设置课表背景图片和背景透明度。
- **主题切换**：支持跟随系统、浅色模式和深色模式。
- **本地存储**：使用 Room 数据库保存课表数据，导入后可离线查看。

## 界面与体验

- 课表卡片采用圆角、描边和主题适配颜色。
- 深色模式下增强了本周课程与非本周课程的对比度。
- 顶部操作按钮、底部导航、查询页、设置页和导入页均做了浅色/深色主题适配。
- 2x2 今日课程小组件使用紧凑信息层级，突出第一节课并展示后续课程信息。
- 自定义课程页面支持“节次 / 具体时间”和“周次 / 日期”两组选择方式。

## 构建

1. 使用 Android Studio 打开项目。
2. 等待 Gradle Sync 完成。
3. 连接真机或启动模拟器。
4. 运行 app 模块。

> 建议直接使用 Android Studio 构建和运行。若本机存在全局 Gradle init 脚本，命令行构建可能会受到本地 Gradle 配置影响。

## 导入原理

1. 通过南京林业大学统一认证系统进行 CAS 登录。
2. 按学校登录流程对密码参数进行加密处理。
3. 登录成功后访问教务系统课表页面。
4. 使用 Jsoup 解析 HTML 表格，提取课程名、教师、教室、周次、节次等信息。
5. 将不连续周次拆分为多条课程详情记录。
6. 写入本地 Room 数据库，并刷新课表和桌面小组件。

## 数据模型

| 表 | 说明 |
|---|---|
| TableBean | 课表元信息，包括名称、学期开始日期、最大周数、显示设置等 |
| CourseBaseBean | 课程基本信息，包括课程名、课程颜色、所属课表 |
| CourseDetailBean | 课程时间详情，包括星期、节次、周次范围、教室、教师、自定义时间等 |

## 小组件

当前提供两类桌面小组件：

- **今日课程列表小组件**：展示当天课程列表，适合较宽区域。
- **今日课程 2x2 小组件**：紧凑展示今日课程，突出第一节课，并显示后续课程或地点信息。

小组件会在课程数据变化、日期变化、时间变化时刷新。

## 说明

本项目仅用于学习和个人课表管理。账号密码仅用于登录南京林业大学统一认证和教务系统，课程数据保存在本地数据库中。
